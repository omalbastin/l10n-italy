Comunicazione IVA (Spesometro) 2017/18

Generate xml file for sending to Agenzia delle Entrate, kwnown as Spesometro.

This module requires PyXB 1.2.4 http://pyxb.sourceforge.net/

Gestisce la Comunicazione periodica IVA con l'elenco delle fatture emesse e ricevute e genera il file da inviare all'Agenzia delle Entrate.

Questo obbligo è conosciuto anche come Spesometro light 2018 e sostistuisce i precedenti obbblighi chiamati Spesometro 2017 e Spesometro.

Il softwware permette di operare in modalità 2017 per rigenerare eventuali file in formato 2017. Per eseguire questa funzione, prima di avviare Odoo eseguire la seguente istruzione:

export SPESOMETRO_VERSION=2.0

Attenzione! Questo modulo è incompatibile con i seguenti moduli OCA:

l10n_it_base versioni [7-11]

l10n_it_fiscalcode versioni [7-11]
